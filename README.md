# Coding Challenge App  

A React Native app designed to help users practice coding exercises in JavaScript, Python, and Java for technical interview preparation.  

## Features  
- Coding challenges with starter code for JavaScript, Python, and Java  
- Scoring system with answer validation  
- Language selection using React Native Picker  
- Interactive UI with KeyboardAvoidingView and Alert  
- Code normalization for consistent answer validation  

## Tech Stack  
- **Framework:** React Native  
- **Languages:** TypeScript, JavaScript  
- **Libraries:** React Native Picker, KeyboardAvoidingView, Alert
